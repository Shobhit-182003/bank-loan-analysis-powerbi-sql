📊 Bank Loan Analysis Dashboard | SQL Server + Power BI
📌 Project Overview

This project is an end-to-end Data Analytics solution developed to analyze bank loan data and provide actionable business insights through interactive dashboards.

The project covers the complete analytics lifecycle, including SQL data extraction, business query development, Power BI data modeling, DAX calculations, and dashboard creation.
